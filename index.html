HTML

  • The head element contains information about the webpage.
  • The body element represents the visisble content shown to the user.
  • !DOCTYPE tells the browser what the type of document the page is when opened.
  • http-equiv holds http data for example, the one in the study guide tells what version of microsoft explorer to be compatible with.
  • Semantic tags are ones used to organize the code and make it more readable like section and body.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • We can assign an HTML tag a class attribute which can then be referenced in the stylesheet to style all elements with that class name.
  • There are 3 ways to style a webpage: inline CSS, internal CSS style sheet, and external CSS style sheet.

Git

  • git status: checks what branch we are currently on
  • git checkout -b "branch name": creates and moves you to that branch.
  • git checkout "branch name": moves you to an already created branch.
  • git pull origin main: pulls the work from the branch named "main" from github to your local machine.
